# 🛒 NexMart – MERN E-Commerce Platform

A full-stack e-commerce application inspired by modern online marketplaces, built using the MERN stack with secure authentication, Stripe payments, admin management, and responsive UI.

---

## 🚀 Live Demo

* Frontend: [https://ecommerce-ci76.vercel.app](https://ecommerce-ci76.vercel.app)
* Backend: [https://ecommerce-xvea.onrender.com](https://ecommerce-xvea.onrender.com)

---

## ✨ Features

### User Features

* Authentication & JWT Authorization
* Product Search & Details
* Add to Cart & Quantity Management
* Stripe Payment Integration
* Coupon System
* Order Placement & Tracking
* Product Reviews
* Responsive UI

### Admin Features

* Product Management
* Order Management
* User Management
* Stock Updates

---

## 🛠️ Tech Stack

### Frontend

* React.js
* React Router DOM
* Axios
* CSS Modules

### Backend

* Node.js
* Express.js
* MongoDB & Mongoose
* JWT Authentication
* Stripe API

### Deployment

* Vercel
* Render
* MongoDB Atlas
* Cloudinary
